Sebelum menjalankan kode ini, pastikan Anda memiliki hal-hal berikut:

- Python 3.x
- pip (Package Installer for Python)

Cara menjalankan:
1. Buat lingkungan virtual:
   python3 -m venv nama_venv
2. Aktifkan lingkungan virtual:
   Di Windows:
	nama_venv\Scripts\activate
   Di macOS/Linux:
	source nama_venv/bin/activate
3. Instal Flask:
   pip install Flask
4. Menjalankan:
   python3 app.py
