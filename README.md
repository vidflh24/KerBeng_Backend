# Overview

## Structure
```
- `AutoPentest/`    # Kelas abstrak sebagai template design pattern yang memanggil masig-masing metode pada direktori `Metode/`
- `CVE/`            # Kumpulan CVE, terdiri dari implementasi template beserta interface (Metode)-nya
- `Metode/`         # Template metode-metode yang digunakan oleh masig-masing CVE
- `utils/`          # Utility/helper function
- `Client.py`       # File utama yang dijalankan
```
